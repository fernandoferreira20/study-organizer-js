# Study Organizer JS

![Status](https://img.shields.io/badge/status-in%20development-yellow) ![Version](https://img.shields.io/badge/version-1.0.0-blue)

A simple Vanilla JavaScript web application for organizing study tasks, subjects, deadlines and progress.

## Demo

GitHub Pages coming soon.

## Features

- Create tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks
- Search tasks
- Sort tasks
- Track progress
- Save tasks with localStorage

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- ES Modules
- localStorage
- Git
- GitHub

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/fernandoferreira20/study-organizer-js.git
   ```
2. Open the project folder in your browser or use a local server.
3. Open `index.html` to start using the app.

## Usage

- Add a new study task using the form.
- Edit, complete, or delete tasks from the task list.
- Use search, filters, and sorting to organize your view.
- Your tasks are stored locally in the browser with `localStorage`.
